# Breast Cancer Diagnosis using Naive Bayes

This project applies a probabilistic machine learning approach to classify breast tumors as **benign or malignant** using diagnostic medical data.

## Dataset
Breast Cancer Wisconsin Dataset (`sklearn.datasets.load_breast_cancer`)

- 569 samples
- 30 clinical diagnostic features
- Target classes: benign and malignant

## Model
Gaussian **Naive Bayes classifier implemented from scratch** in Python.

## What I Implemented
- Prior probability calculation
- Gaussian likelihood estimation
- Naive Bayes prediction logic
- Train/test split for evaluation

## Tools
Python, NumPy, scikit-learn, Matplotlib

## Run the Project
Open the notebook and run all cells:

`BreastCancer_Naive_Bayes.ipynb`
